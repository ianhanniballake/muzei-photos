<b>Muzei - Photos album</b> is a plugin for <a href="https://muzei.co/">Muzei</a>, which is an Android live wallpaper app, by Roman Nurik and Ian Lake.

This plugin allow you to pick one <a href="https://www.google.com/photos/about/">Google Photos</a> album as a wallpaper source for the <a href="https://muzei.co/">Muzei</a> app, so that your favorite photos can be displayed as wallpaper on both the home and the lock screen.

In order to use this plugin, you will have to:
- Install the <b>Muzei</b> app first
- Authorize the plugin to access your <b>Google Photos</b> library

It's open-source: <a href="https://github.com/xinthink/muzei-photos">github.com/xinthink/muzei-photos</a>
