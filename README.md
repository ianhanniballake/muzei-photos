# Muzei - Google Photos

[![Build Status][ci-badge]][ci-link]
[![MIT][license-badge]][license]

This's a plugin for [Muzei], which is an Android live wallpaper app. This plugin allows you making [Google Photos] a [Muzei] wallpaper source.
